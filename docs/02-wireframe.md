# 画面設計（ワイヤー）

## 画面全体
- 1ページ構成（PC/モバイル共通）。
- 背景は淡いグラデーション、メインはカードレイアウト。
- 主要導線は「入力 → 一覧 → フィルタ/ソート」。

## ヘッダー
- 左: アプリ名 `Task Manager`
- 右: 現在日付（任意）
- 下段: フィルタ（すべて / 未完了 / 完了）、ソート（作成日 / 期限 / 優先度）

## タスク入力フォーム
- タイトル（必須）
- 説明（任意）
- 期限（任意: date）
- 優先度（任意: low / medium / high）
- 追加ボタン（主要CTA）

## タスク一覧
- カードリスト
- カード内要素
  - 左: 完了トグル
  - 中: タイトル、説明
  - 右: 期限、優先度バッジ、削除ボタン
- インライン編集
  - タイトル/説明はクリックで編集モード
  - 期限/優先度も同一カード内で編集可能

## フッター（任意）
- 件数表示（合計 / 未完了）
- ローカル保存のステータス（任意）
